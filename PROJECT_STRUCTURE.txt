DATABASE TABLES (9 tables)
===========================

1. majors
   └── Academic majors/departments

2. users
   └── System users (students, instructors, admins)
   └── FK: major_id → majors

3. courses
   └── Course information
   └── FK: instructor_id → users

4. enrollments
   └── Student course enrollments with progress tracking
   └── FK: student_id → users
   └── FK: course_id → courses

5. content
   └── Course materials (lectures, videos, assignments, etc.)
   └── FK: uploader_id → users
   └── FK: course_id → courses

6. ratings
   └── User ratings on content (1-5 stars)
   └── FK: user_id → users
   └── FK: content_id → content

7. comments
   └── User comments on content
   └── FK: user_id → users
   └── FK: content_id → content

8. downloads
   └── Content download tracking
   └── FK: user_id → users
   └── FK: content_id → content

9. admin_actions
   └── Administrative action logs
   └── FK: admin_id → users

SETUP INSTRUCTIONS
==================

Method 1: Automated Setup (Recommended)
----------------------------------------
1. Initialize Git:
   > setup_git.bat

2. Setup Database:
   > setup_database_complete.bat
   (Follow prompts to enter MySQL credentials)

3. Configure:
   > copy config\database.example.php config\database.php
   (Edit with your credentials)


Method 2: Manual Setup
----------------------
1. Git:
   > git init
   > git add .
   > git commit -m "Initial commit"

2. Database:
   > mysql -u root -p -e "CREATE DATABASE tsharok CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"
   > mysql -u root -p tsharok < database\schema_complete.sql
   > mysql -u root -p tsharok < database\seed_data_complete.sql

3. Configure:
   Copy and edit config\database.example.php
